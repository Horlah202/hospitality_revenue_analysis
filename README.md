# Hospitality Revenue Analysis

## Project Overview
This project analyzes 6 months of hotel and short-stay apartment data to understand:
- Revenue drivers
- Occupancy trends
- Pricing performance

## Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

## Key Insights
- High occupancy does not always mean high revenue
- Weekends outperform weekdays
- Some room types are underpriced

## How to Run
1. Clone the repository
2. Open the notebook in Jupyter
3. Run cells from top to bottom
